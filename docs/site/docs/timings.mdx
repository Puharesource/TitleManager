Timings determine how long text elements, such as titles or frames in an animation, appear on the screen. This includes
how long a title stays visible and the duration of fade-in and fade-out transitions.

:::note
When used outside of titles, the `fadeIn` and `fadeOut` values function as the `stay` value.
:::

## Key Concepts

### What Are Timings?

**Timings control:**

These values are expressed in **ticks**, where one tick equals 1/20th of a second.

### Usage Examples

Timings can be applied to:

- Title messages
- Animated frames
- Any text element managed by TitleManager
