title: Message Title Command
---

Sends a title message to a specified player. You can use `<nl>`, `{nl}`, or `%\nl%` in the message to include a
subtitle.

## Syntax

```
/tm msg [parameters] <player> <message>
/tm message [parameters] <player> <message>
```

## Arguments

- **`[parameters]`**: Optional parameters to customize the message.
- **`<player>`**: The player to whom the message will be sent.
- **`<message>`**: The content of the title message.

## Supported Parameters

- **`-silent`**: Suppresses any response text from the command.
- **`-fadein=ticks`, `-fade-in=ticks`**: Specifies the duration in ticks for the title to fade onto the screen. Example: `-fadein=20` sets the fade-in time to 20 ticks (1 second).
- **`-stay=ticks`**: Specifies the duration in ticks for the title to remain visible. Example: `-stay=40` sets the stay time to 40 ticks (2 seconds).
- **`-fadeout=ticks`, `-fade-out=ticks`**: Specifies the duration in ticks for the title to fade off the screen. Example: `-fadeout=20` sets the fade-out time to 20 ticks (1 second).
