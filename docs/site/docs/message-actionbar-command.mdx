title: Message Actionbar Command
---

Sends an actionbar message to the specified player.

## Syntax

```
/tm amsg [parameters] <player> <message>
/tm amessage [parameters] <player> <message>
```

## Arguments

- **`[parameters]`**: Optional parameters to modify the behavior of the command.
- **`<player>`**: The player to whom the actionbar message will be sent.
- **`<message>`**: The content of the actionbar message to be displayed.

## Supported Parameters

- **`-silent`**: Suppresses any response text from the command.
