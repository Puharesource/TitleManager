title: Permissions
---

TitleManager command permissions default to server operators.

## Command permissions

| Permission                                 | Command                                                                             |
| ------------------------------------------ | ----------------------------------------------------------------------------------- |
| `titlemanager.command.version`             | `/tm version`, `/tm v`                                                              |
| `titlemanager.command.reload`              | `/tm reload`                                                                        |
| `titlemanager.command.diagnostics`         | `/tm diagnostics`, `/tm diag`                                                       |
| `titlemanager.command.animations`          | `/tm animations`                                                                    |
| `titlemanager.command.scripts`             | `/tm scripts`                                                                       |
| `titlemanager.command.sidebar.toggle`      | `/tm sidebar toggle`, `/tm sb toggle`, `/tm scoreboard toggle`                      |
| `titlemanager.command.playerlist.toggle`   | `/tm playerlist toggle`, `/tm pl toggle`                                            |
| `titlemanager.command.title.broadcast`     | `/tm broadcast <message>`, `/tm bc <message>` under title commands                  |
| `titlemanager.command.title.message`       | `/tm msg <player> <message>`, `/tm message <player> <message>` under title commands |
| `titlemanager.command.title.toggle`        | welcome title toggle command                                                        |
| `titlemanager.command.actionbar.broadcast` | `/tm actionbar broadcast <message>`, `/tm actionbar bc <message>`                   |
| `titlemanager.command.actionbar.message`   | `/tm actionbar msg <player> <message>`, `/tm actionbar message <player> <message>`  |
| `titlemanager.command.actionbar.toggle`    | welcome actionbar toggle command                                                    |

## Notification permissions

| Permission                   | Description                                                                              |
| ---------------------------- | ---------------------------------------------------------------------------------------- |
| `titlemanager.update.notify` | Receives update notifications when update checks are enabled and an update is available. |
