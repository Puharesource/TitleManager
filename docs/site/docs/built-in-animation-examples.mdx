These examples render locally in Docusaurus using the same TitleManager Kotlin/JS web library used by the web viewer. They do not embed an external site.

The examples use the default `sunset` gradient from `gradients.yml` where a named gradient is required.

## `shine`

```text
${shine:[0;2;0][0;25;0][0;25;0][&3;&b]My Server}
```

<AnimationPreview
  title={"shine"}
  text={"${shine:[0;2;0][0;25;0][0;25;0][&3;&b]My Server}"}
  scale={50}
/>
