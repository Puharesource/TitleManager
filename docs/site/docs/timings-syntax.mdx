title: Timings Syntax
---

You can define timings in two different ways: the **full syntax** and the **short syntax**.

## Syntax in Action

### Full Syntax Example

```
[5;60;5]
```

- **5 ticks** to fade in
- **60 ticks** to stay visible
- **5 ticks** to fade out

### Short Syntax Example

```
[40]
```

- **40 ticks** to stay visible
- **fadeIn** and **fadeOut** default to 0 ticks
